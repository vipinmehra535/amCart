## About

AMCart is an Open-source E-commerce application built with Flutter for Android and iOS.

## Features

- User authentication (sign up, log in, reset password)
- Product listings with pagination
- Product details
- Add/remove products to/from cart
- Checkout process
- Order history
- Search for products
